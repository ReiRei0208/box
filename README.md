# 🥳 Welcome

## BlocksBox

BlocksBox是一个一个**简单、易用、方便、快捷**的 TurboWarp扩展分享平台，包括 团队、个人 制作的扩展，并**支持投稿 原创扩展、汉化扩展**

## 如何使用

复制扩展URL到 **TurboWarp桌面端beta.2** 或更高的版本，或者 [**在线加载**](https://turbowarp.org/editor)****
